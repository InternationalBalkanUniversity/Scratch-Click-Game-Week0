# Grading Rubric (10 Points Total)

## Functionality (8 Points)

| Criteria | Points | Description |
|----------|--------|-------------|
| Game starts on green flag click | 2 | The game initializes properly when the green flag is clicked |
| Score increases on sprite click | 2 | Clicking the sprite correctly increments the score counter |
| 10-second timer implementation | 2 | The game runs for exactly 10 seconds and stops |
| "Game Over" message with score | 2 | The sprite displays "Game Over! Your score is: X" with the correct final score |

## Code Quality (1 Point)

| Criteria | Points | Description |
|----------|--------|-------------|
| Code organization | 1 | Scripts are well-organized and easy to understand |

## Submission (1 Point)

| Criteria | Points | Description |
|----------|--------|-------------|
| Proper file submission | 1 | .sb3 file is correctly uploaded to the GitHub repository |
