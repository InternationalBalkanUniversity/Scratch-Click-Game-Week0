# How to Submit

Follow these steps to submit your Scratch Click Game assignment:

## Step 1: Accept the GitHub Classroom Link
- Click on the GitHub Classroom assignment link provided by your instructor
- This will create your personal repository for the assignment

## Step 2: Create Your Game in Scratch
- Go to [Scratch](https://scratch.mit.edu/)
- Build your Click Game according to the requirements in README.md
- Test your game to ensure all features work correctly:
  - Green flag starts the game
  - Clicking the sprite increases the score
  - Timer runs for exactly 10 seconds
  - "Game Over! Your score is: X" message appears at the end

## Step 3: Download Your Project
- In Scratch, click on **File** → **Save to your computer**
- This will download a `.sb3` file to your computer

## Step 4: Upload to GitHub Repository
- Navigate to your GitHub repository (the one created from the Classroom link)
- Click on **Add file** → **Upload files**
- Drag and drop your `.sb3` file or click to browse and select it
- Add a commit message (e.g., "Add Click Game project")

## Step 5: Commit Your Changes
- Scroll down and click the **Commit changes** button
- Your assignment is now submitted!

## Important Notes
- Make sure your `.sb3` file is named appropriately (e.g., `ClickGame.sb3`)
- Submit before the deadline
- You can update your submission by uploading a new version before the deadline
